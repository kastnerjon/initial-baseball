# Lessons Learned

Add corrections here when the PM or QA catches a mistake.

Current locked decisions:

- Use `AGENTS.md` as source-of-truth for agents.
- Keep `CLAUDE.md` as tiny compatibility redirect only.
- No randomized hint order by default.
- Game settings use proposal/counterproposal/acceptance.
- Default alpha: 3 innings, 3 strikes, ghost runner ON.
- Hint label is `Main decade played in`, not `Era`.
- Stats hint is configurable through Custom Stats Picker.
- If Baseball Reference WAR is used, label it `bWAR`.
- Pitcher hints pre-populate from DB but are editable before submission.
- Random opponents and game-only chat are alpha scope.
- League Lite is alpha scope.
- Free app; no payments or ads in alpha.
